Restauração Manual

Passo a passo:

1. Baixar: rclone copy {{ config.rclone_destino }}/ARQUIVO.gpg /tmp/

2. Descriptografar: gpg --batch --passphrase "SENHA" --decrypt -o /tmp/backup.tar.zst /tmp/ARQUIVO.gpg

3. Descomprimir: zstd -d /tmp/backup.tar.zst && tar -xf /tmp/backup.tar -C /tmp/restore/

4. Banco: sudo -u postgres psql beneficios_db < /tmp/restore/banco.sql

5. Docs: cp -r /tmp/restore/documentos/* /var/www/sistema_beneficios_data/media/

6. Restart: service beneficios restart
